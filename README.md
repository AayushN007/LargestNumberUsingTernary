# Largest Number Using Ternary Operator

## 📘 Overview

This Python program demonstrates the use of the **Ternary (Conditional) Operator** to compare two numbers.

The program accepts two integers from the user and determines whether the first number is greater, the second number is greater, or both numbers are equal—all using a single conditional expression.

---

## ✨ Features

* Accepts two integer inputs from the user.
* Compares the values using the ternary operator.
* Displays the larger number or indicates if both numbers are equal.
* Demonstrates concise decision-making in Python.

---

## 🛠️ Concepts Covered

* User Input (`input()`)
* Type Conversion (`int()`)
* Ternary (Conditional) Operator
* Comparison Operators (`>`, `==`)
* Decision Making

---

## 📂 Program Workflow

1. Read two integers from the user.
2. Compare the values using nested ternary operators.
3. Display:

   * **A is Greater** if `a > b`
   * **B is Greater** if `b > a`
   * **A and B are equal** if both values are the same
